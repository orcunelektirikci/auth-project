# Nuxt3 Starter Kit with Laravel Passport or Sanctum Auth

### Uses: 

- Pinia
- Persisted State
- Typescript
- Bearer Auth Token and/or CSRF Tokens
- Multi-language support
- Tailwind
- Admin Panel And Frontend (WIP)
- SEO support (WIP)
- Unit Tests (WIP)
- Nuxt UI (WIP)
